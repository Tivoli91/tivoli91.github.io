终于要开始讲script了，我们将之前所讲过的各种知识点保存在特定后缀名的文件里就是一个powershell script了，所以我的理解script本身也是一个scriptblock。

后缀名

powershell有以下几种后缀名，分别对应不同的应用场景，目前我们只要用到.ps1这个，以后的碰到了再细说。

文件扩展名(File extensions)

  • PS1 - Windows PowerShell外壳脚本
  • PSD1 - Windows PowerShell数据文件
  • PSM1 - Windows PowerShell模块文件
  • PS1XML - Windows PowerShell格式和类型定义
  • CLIXML - Windows PowerShell序列化数据
  • PSC1 - Windows PowerShell控制台文件
  • PSSC - Windows PowerShell会话配置文件
